рІГрѕЁ рЇЋрѕ«рїђріГрЅх рЅарѕЮрѕхрѕЇ рѕІрІГ рѕіріќрѕГ рІерѕџрЅйрѕѕрІЇріЋ рІерЅєрІ│ рЅарѕйрЅ│ рѕѕрѕўрѕѕрІерЅх рІерЅ░рѕ░рѕФ ріљрІЇрЇб рЅаDINOv2 рѕърІ┤рѕЇ рѕІрІГ рЅ░рѕўрѕхрѕГрЅХ рІерЅєрІ│ рѕЮрѕхрѕјрЅйріЋ рЅарѕўрїарЅђрѕЮ рІерЅарѕйрЅ│рІЇріЋ ріарІГріљрЅх рѕѕрѕўрЅ░ріЋрЅарІГ рІГрѕерІ│рѕЇрЇб

## рѕўрїЇрЅбрІФ

рІГрѕЁ рѕўрЅ░рїЇрЅарѕфрІФ рЅ░рїарЅЃрѕџрІјрЅй рІерЅєрІ│ рѕЮрѕхрѕЇ ріЦріЋрІ▓рѕ░рЅЁрѕЅ ріЦріЊ рѕіріќрѕГ рІерѕџрЅйрѕѕрІЇріЋ рІерЅєрІ│ рЅарѕйрЅ│ рЅхріЋрЅарІФ ріЦріЋрІ▓рІФрїѕріЎ рІФрѕхрЅйрѕІрѕЇрЇб рЅхріЋрЅарІФрІЇ рІерѕџріФрѕёрІ░рІЇ рЅарЅЁрІхрѕў-рѕхрѕЇрїаріЊ рЅарЅ░рѕ░рїарІЇ ріЦріЊ рѕѕрЅєрІ│ рЅарѕйрЅ│рІјрЅй рЅарЅ░рѕхрЅ░ріФріерѕѕ (fine-tuned) рІеDINOv2 рѕърІ┤рѕЇ ріљрІЇрЇб ріерЅ░рЅ░ріљрЅарІерІЇ рІерЅарѕйрЅ│ рѕхрѕЮ рЅарЅ░рїерѕЏрѕф рІЮрѕГрІЮрѕГ рѕўрїЇрѕѕрїФ ріЦріЊ рІерѕџрѕўріерѕЕ рІерѕІрЅЦрѕФрЅХрѕф рѕЮрѕГрѕўрѕФрІјрЅй (рѕўрѕерїЃрІЇ ріФрѕѕ) рІГрѕ░рїБрѕЇрЇб

## рѕўрѕхрЇѕрѕГрЅХрЅй

рЇЋрѕ«рїђріГрЅ▒ріЋ рѕѕрѕЏрѕхрігрІх рІерѕџріерЅ░рѕЅрЅх рІФрѕхрЇѕрѕЇрїІрѕЅрЇд

* Python 3.x
* Streamlit
* PyTorch
* Transformers library (Hugging Face)
* Pillow (PIL)

ріЦріљрІџрѕЁріЋ рѕўрѕхрЇѕрѕГрЅХрЅй рѕѕрѕўрїФріЋ `pip` рѕўрїарЅђрѕЮ рІГрЅйрѕІрѕЅрЇд

```
pip install streamlit torch transformers pillow

```

## installation

1. рІГрѕЁріЋріЋ рѕфрЇќрІџрЅХрѕф ріГрѕјріЋ рІФрІхрѕГрїЅрЇд
   ```
   git clone [рІерѕфрЇќрІџрЅХрѕфрІј_рІЕріарѕГріцрѕЇ]
   cd Skin-Disease-Classifier-Project

   ```
2. рІерѕџрІФрѕхрЇѕрѕЇрїЅрЅхріЋ рѕІрІГрЅЦрѕерѕфрІјрЅй рІГрїФріЉ (ріерѕІрІГ рІГрѕўрѕЇріерЅ▒)рЇб
3. рІФрѕ░рѕѕрїаріЉрЅхріЋ рІѕрІГрѕЮ рѕѕрѕўрїарЅђрѕЮ рІФрѕ░рЅАрЅхріЋ рІеDINOv2 рѕърІ┤рѕЇ [path/to/your/finetuned/dinov2/model] рЅарѕџрѕѕрІЇ рЅдрЅ│ рІФрѕхрЅђрѕЮрїАрЇб

## ріарїарЅЃрЅђрѕЮ

рѕўрЅ░рїЇрЅарѕфрІФрІЇріЋ рѕѕрѕЏрѕхрігрІх рІерѕџріерЅ░рѕѕрІЇріЋ рЅхрІЋрІЏрІЮ рІГрїарЅђрѕЎрЇд

```
streamlit run skin.py

```

(рІерІІріЊрІЇ рІерЇЊрІГрІўріЋ рЇІрІГрѕЇ рѕхрѕЮ `skin.py` ріљрІЇ)

рІГрѕЁ рЅхрІЋрІЏрІЮ рІерІхрѕГ ріарѕ│рѕй рІГріерЇЇрЅ│рѕЇ ріЦріЊ рѕўрЅ░рїЇрЅарѕфрІФрІЇріЋ рІФрѕ│рІФрѕЇрЇб

1. "рѕЮрѕхрѕЇ рІГрѕЮрѕерїА..." рІерѕџрѕѕрІЇріЋ рЅЂрѕЇрЇЇ рЅарѕўрїФріЋ рІерЅєрІ│ рѕЮрѕхрѕЇ рІГрѕхрЅђрѕЅрЇб
2. рѕЮрѕхрѕЅ ріерЅ░рѕ░рЅђрѕѕ рЅаріІрѕІ рѕўрЅ░рїЇрЅарѕфрІФрІЇ рЅарѕФрѕх-рѕ░рѕГ рЅхріЋрЅарІФрІЇріЋ рІФріеріЊрІЇріЊрѕЇрЇб
3. рІерЅ░рЅ░ріљрЅарІерІЇ рІерЅарѕйрЅ│ рѕхрѕЮрЇБ рІЮрѕГрІЮрѕГ рѕўрїЇрѕѕрїФ ріЦріЊ рІерѕџрѕўріерѕЕ рІерѕІрЅЦрѕФрЅХрѕф рѕЮрѕГрѕўрѕФрІјрЅй (рѕўрѕерїЃрІЇ ріФрѕѕ) рІГрЅ│рІФрѕЇрЇб

рІерѕўрЅ░рїЇрЅарѕфрІФрІЇ рІІріЊ рїѕрїй рЅЁрїйрЅарЅ│рІі рїѕрїй ріЦрІГрЅ│

## рѕхрѕѕ DINOv2 рѕърІ┤рѕЇ ­ЪДа

рІГрѕЁ рЇЋрѕ«рїђріГрЅх рІерЅєрІ│ рЅарѕйрЅ│рІјрЅйріЋ рѕѕрѕўрѕѕрІерЅх **DINOv2** рІерЅ░рЅБрѕѕрІЇріЋ рІері«рѕЮрЇњрІЕрЅ░рѕГ рЅфрІЦріЋ рѕърІ┤рѕЇ рІГрїарЅђрѕЏрѕЇрЇб DINOv2 рЅаMeta AI рІерЅ░рѕ░рѕФ рѕ▓рѕєріЋ рЅарЅЦрІЎ рѕЮрѕхрѕјрЅй рѕІрІГ рІерѕ░рѕѕрїаріљ рЅарѕўрѕєріЉ рѕѕрЅ░рѕѕрІФрІЕ рІерЅфрІЦріЋ рЅ░рїЇрЅБрѕФрЅх рїарЅЃрѕџ рІерѕєріЉ рЅБрѕЁрѕфрІФрЅхріЋ рѕўрѕЏрѕГ рІГрЅйрѕІрѕЇрЇб ріЦрІџрѕЁ рѕІрІГ рѕѕрЅєрІ│ рЅарѕйрЅ│ рѕўрѕѕрІФ рЅарЅ░рѕѕрІе рѕўрѕЇріЕ ріЦріЋрІ▓рѕ░рѕФ рЅ░рІ░рѕГрїј рЅ░рѕхрЅ░ріФріГрѕЈрѕЇрЇб рѕхрѕѕ DINOv2 рЅ░рїерѕЏрѕф рѕўрѕерїЃ рЅаHugging Face [ріЦрІџрѕЁ](https://huggingface.co/facebook/dinov2-base "null") рІФрїѕріЏрѕЅрЇб

## рЇЋрѕ«рїђріГрЅ▒ ріЦріЋрІ┤рЅх рІГрѕ░рѕФрѕЇ? ­ЪЏа№ИЈ

1. **рѕЮрѕхрѕЇ рѕўрѕхрЅђрѕЇ:** рЅ░рїарЅЃрѕџрІЇ рІерЅєрІ│рІЇріЋ рѕЮрѕхрѕЇ рЅарѕўрЅ░рїЇрЅарѕфрІФрІЇ рѕІрІГ рІГрѕ░рЅЁрѕІрѕЇрЇб
2. **рЅЁрІхрѕў-рІЮрїЇрїЁрЅх:** рІерЅ░рѕ░рЅђрѕѕрІЇ рѕЮрѕхрѕЇ рѕърІ┤рѕЅ рѕірѕерІ│рІЇ рЅарѕџрЅйрѕЇ рЅЁрѕГрїИрЅх рІГрІўрїІрїЃрѕЇрЇб
3. **рЅхріЋрЅарІФ:** рЅЁрІхрѕў-рІЮрїЇрїЁрЅх рІерЅ░рІ░рѕерїѕрІЇ рѕЮрѕхрѕЇ рІѕрІ░ DINOv2 рѕърІ┤рѕЇ рІГрѕІріФрѕЇ ріЦріЊ рѕърІ┤рѕЅ рѕіріќрѕГ рІерѕџрЅйрѕѕрІЇріЋ рІерЅєрІ│ рЅарѕйрЅ│ рІГрЅ░ріљрЅЦрІФрѕЇрЇб
4. **рІЇрїцрЅх рѕЏрѕ│рІерЅх:** рѕўрЅ░рїЇрЅарѕфрІФрІЇ рІерЅ░рЅ░ріљрЅарІерІЇріЋ рІерЅарѕйрЅ│ рѕхрѕЮрЇБ ріарїГрѕГ рѕўрїЇрѕѕрїФ ріЦріЊ рѕірІФрѕхрЇѕрѕЇрїЅ рІерѕџрЅйрѕЅ рІерѕІрЅЦрѕФрЅХрѕф рѕЮрѕГрѕўрѕФрІјрЅйріЋ рІФрѕ│рІФрѕЇрЇб

## рІЇрїцрЅХрЅ╣ рѕЮріЋ рІФрѕЁрѕЇ рЅхріГріГрѕЇ ріЊрЅИрІЇ? РюЁ

рІГрѕЁ рЇЋрѕ«рїђріГрЅх рІерѕЁріГрѕЮріЊ рЅБрѕѕрѕЎрІФрІјрЅйріЋ рѕхрѕФ рѕѕрѕЏрїѕрІЮ ріЦріЊ рѕіріќрѕГ рІерѕџрЅйрѕѕрІЇріЋ рІерЅєрІ│ рЅарѕйрЅ│ рЅарЇЇрїЦріљрЅх рѕѕрѕўрѕѕрІерЅх ріЦріЋрІ▓рѕерІ│ рЅ│рѕхрЅд рІерЅ░рѕ░рѕФ рѕўрѕ│рѕфрІФ ріљрІЇрЇб рІерІџрѕЁ рЇЋрѕ«рїђріГрЅх рЅхріЋрЅарІФрІјрЅй рЅарѕ░рѕѕрїаріљрІЇ рІеDINOv2 рѕърІ┤рѕЇ ріарЇѕрї╗рїИрѕЮ рѕІрІГ рІерЅ░рѕўрѕ░рѕерЅ▒ ріЊрЅИрІЇрЇб рѕърІ┤рѕЅ рЅарїЦрѕЕ рѕЂріћрЅ│ рІерѕ░рѕѕрїаріљ ріерѕєріљ рЅхріГріГрѕѕріЏ рІерѕўрѕєріЋ ріЦрІхрѕЅ ріерЇЇрЅ░ріЏ ріљрІЇрЇб **рѕєріќрѕЮрЇБ рІГрѕЁ рѕўрЅ░рїЇрЅарѕфрІФ рѕѕрѕЁріГрѕЮріЊ рѕЮріГрѕГ рІѕрІГрѕЮ рѕѕрѕЎрІФрІі рѕЮрѕГрѕўрѕФ рѕЮрЅхріГ ріарІГрІ░рѕѕрѕЮрЇб** рѕѕрѕЏріЋріЏрІЇрѕЮ рІерЅєрІ│ рѕЂріћрЅ│ рЅхріГріГрѕѕріЏ рѕЮрѕГрѕўрѕФ ріЦріЊ рѕЁріГрѕЮріЊ рѕѕрѕЏрїЇріўрЅх рѕЂрѕЇрїірІю рЅЦрЅЃрЅх рІФрѕѕрІЇ рІерѕЁріГрѕЮріЊ рЅБрѕѕрѕЎрІФ рѕЏрѕЏріерѕГ ріарѕхрЇѕрѕІрїі ріљрІЇрЇб

## рІерЇЋрѕ«рїђріГрЅх рѕЏрЅарѕЇрїИрїірІФ ­Ъџђ

рІГрѕЁріЋріЋ рЇЋрѕ«рїђріГрЅх рІерЅарѕѕрїа рѕѕрѕЏрѕ╗рѕ╗рѕЇ ріЦріЊ рѕѕрѕЏрѕЇрѕЏрЅх рІерѕџріерЅ░рѕЅрЅхріЋ ріљрїЦрЅдрЅй рїЇрѕЮрЅх рІЇрѕхрїЦ рѕЏрѕхрїѕрЅБрЅх рІГрЅ╗рѕІрѕЇрЇд
